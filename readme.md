Once cloned/installed just run the "MasterV2" file inside. all others are just functionality that do not need to be touched.
There are two tools needed to run this program

1. AHK (Autohotkey)
This is what the scripts are built on.
```https://www.autohotkey.com/download/ahk-v2.exe```.

2. Git
Git is used to download and update the scripts. It's not absolutly nessesary but it may scream at you without it
```https://github.com/git-for-windows/git/releases/download/v2.54.0.windows.1/Git-2.54.0-64-bit.exe```


There are two ways to install. 
#1 Goto https://github.com/Chariot-UHCW/AutoHotkey-Scripts and Click the green "<> Code" button, then clicking "Download Zip". Once downloaded right click it and "extract". 

#2 Run this in the terminal (git required)
```git clone https://github.com/josh-white-uhcw/uhcw-ahk.git```.

Once done enter the folder and run MasterV2.
Updates are handeled automatically if git is installed, if not you will need to download manually.
